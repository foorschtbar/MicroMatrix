# SK6805EC15-Matrix

A very tiny 8x8 Pixel LED Matrix with 64 SK6805-EC15 LEDs. With four panels in a chain (8x32 Pixel) perfect for the [PixelIt](https://github.com/o0shojo0o/PixelIt) project.
